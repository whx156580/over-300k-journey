# 🤖 AI 模型与评测 (AI-Evaluation)

> **所属模块**: 软件测试模块 (Software Testing)
> **专注领域**: LLM 评测、RAG 评估指标、AI Agent 评测。

## 🛠️ 技术栈分类 (Tech Stacks)

- **[ragas/](./ragas)**: RAG 应用的自动化评估框架。
- **[deepeval/](./deepeval)**: LLM 应用的单元测试与评估。

## 📅 计划安排

### 第一阶段：AI 基础与 Prompt (Day 1-30)
- **Day 1-10**: Prompt 工程测试策略、对抗性 Prompt。
- **Day 11-20**: 主流评测框架 (Ragas/TruLens/DeepEval)。
- **Day 21-30**: 数据集构造 (MMLU/CMMLU/自定义域数据集)。

### 第二阶段：RAG 与 Agent (Day 31-70)
- **Day 31-50**: RAG 三元组评估 (忠实度、相关性、答案相关性)。
- **Day 51-60**: 检索阶段评估 (Recall/Precision/Hit Rate)。
- **Day 61-70**: Agent 任务规划与工具调用评测。

### 第三阶段：工程化与规模化 (Day 71-100)
- **Day 71-85**: 自动化评测流水线集成。
- **Day 86-95**: 模型幻觉检测与内容安全。
- **Day 96-100**: AI 评测专家面试题。

## 📊 验收标准
1. 完成 RAG 全生命周期评估报告。
2. 自研模型幻觉自动检测脚本。
3. 建立 1000+ Case 的测试基准集。
