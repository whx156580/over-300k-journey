# 🎨 UI 自动化测试 (UI-Testing)

> **所属模块**: 软件测试模块 (Software Testing)
> **专注领域**: Web/App UI 自动化测试、视觉回归、用户体验监控。

## 🛠️ 技术栈分类 (Tech Stacks)

本项目按技术栈对代码进行隔离管理：

- **[playwright/](./playwright)**: Playwright 脚本、POM 模型与环境配置。
    - 📄 [环境搭建与初探 (Basic)](./playwright/playwright_env_setup.md)
- **[selenium/](./selenium)**: Selenium 脚本、驱动管理与传统测试方案。

## 📅 计划安排

### 第一阶段：UI 自动化核心 (Day 1-30)
- **Day 1-10**: 现代选择器策略、Shadow DOM 处理、复杂交互模拟。
- **Day 11-20**: 网络请求拦截 (Mocking)、鉴权状态复用。
- **Day 21-30**: 视觉回归测试 (Visual Regression) 与多端适配。
