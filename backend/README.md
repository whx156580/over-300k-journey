# ⚙️ 后端开发模块 (Backend Development)

> **核心目标**: 设计高并发、高可用的分布式后台架构，掌握服务端核心技术。

## 📁 目录结构
- **[distributed/](./distributed)**: 微服务架构、Kafka 消息队列、Redis 分布式锁。
- **[database/](./database)**: MySQL 调优、分库分表、事务一致性。
- **[system-design/](./system-design)**: 高可用架构、可伸缩性设计、限流熔断。

## 🛠️ 技术栈
- **语言**: Go, Java (Spring Boot).
- **存储**: MySQL, Redis, MongoDB, ElasticSearch.
- **中间件**: Kafka, RocketMQ, RabbitMQ.
- **进阶**: K8s 治理, Service Mesh, DDD (领域驱动设计).
