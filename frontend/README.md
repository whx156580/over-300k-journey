# 🎨 前端开发模块 (Frontend Development)

> **核心目标**: 构建高性能、可维护的 UI 系统，深度掌握前端工程化与性能优化。

## 📁 目录结构
- **[frameworks/](./frameworks)**: React, Vue3, TypeScript 深度实践。
- **[engineering/](./engineering)**: Vite, Webpack, CI/CD 流水线配置。
- **[performance/](./performance)**: Lighthouse 性能指标、Web Vitals 优化。

## 🛠️ 技术栈
- **核心**: React, Vue, TypeScript, Tailwind CSS.
- **工具**: Vite, Jest, ESLint, Prettier.
- **进阶**: 微前端 (qiankun), WebAssembly, 视觉回归。
