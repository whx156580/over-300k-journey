# 🤖 AI 开发模块 (AI Development)

> **核心目标**: 落地 AI 应用，通过智能化手段赋能业务与效率。

## 📁 目录结构
- **[llm-agent/](./llm-agent)**: 大模型应用、Prompt 工程与 Agent 开发。
- **[mlops/](./mlops)**: 模型部署、监控与 MLOps 工程化。
- **[dataset/](./dataset)**: 数据管道、特征平台与数据集管理。

## 🛠️ 技术栈
- **核心**: LangChain, LlamaIndex, PyTorch, TensorFlow.
- **应用**: OpenAI API, HuggingFace, Milvus (向量数据库).
- **进阶**: 模型微调 (SFT), Rerank, 智能化自愈。
